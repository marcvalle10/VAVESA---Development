Instrucciones generales (para todos los equipos):
Formen equipos de 3 integrantes.

Elijan una de las siguientes temáticas.

Creen un sitio web que contenga:

Página principal con título y encabezado del evento.

Imagen o fondo alusivo al tema.

Texto principal con la invitación formal o mensaje del evento.

Música o audio de fondo.

Contador regresivo con JavaScript (cuántos días faltan para el evento).

Sección de confirmación de asistencia (formulario con nombre, correo y botón de enviar).

Efectos visuales con CSS o JS.

Footer con los créditos del equipo (nombres, grupo, fecha).

Incorporen al menos una funcionalidad con JavaScript:

Contador de días faltantes.

Validación de formulario.

Botón de “Confirmar asistencia” con mensaje dinámico.

Efecto de animación o cambio de color automático.

Guarden el proyecto completo  y suban una captura de pantalla o grabación corta mostrando el funcionamiento.





INVITACIÓN 4: Apertura de un local
Instrucciones específicas:

Diseño moderno con colores del local.

Incluye logo o nombre comercial.

Texto breve invitando a la apertura, promociones o sorpresas.

Botón con animación de “Ver ubicación”.

Música ambiental o instrumental moderna.

Contador de días para la apertura.

Sección para confirmar asistencia o registro de invitados.

(Opcional) Agregar un efecto con JS: confeti o animación de apertura.
